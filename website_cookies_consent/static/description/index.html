Configuration
Choosing a consent manager
Go to the "Website" - "Configuration" - "Settings" menu and select a CMP in the "Privacy" - "Consent Manager" section.
The "Odoo Cookie Bar" value is related to the Cookie Bar setting. Therefore, if you change a value of the Consent Manager, it will affect the Cookie Bar settings.
